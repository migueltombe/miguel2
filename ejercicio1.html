red mundial

en informatica, la red mundial (www) o red informatica mundial cumunmente cocnida como la web, es un sistema de distribucion de ducumentos de hepertexto o hipermedio interconectados y accesibles via internet. con un navegador web, un usuario visual sitio web compuesto de pagina web que pueden contener texto, imagen, texto, u otros contenidos multimedia, y navegar atravez de esa pagina usando hiperenlaces..
